EasyOCR is a user-friendly OCR library with excellent accuracy and broad language support. Mozo provides 5 pre-configured variants optimized for different use cases.

## The Problem

Text extraction from images is essential for document processing, but traditional OCR solutions require complex setup, language-specific configurations, and framework expertise.

EasyOCR eliminates this complexity with simple, ready-to-use models that support 80+ languages out of the box.

## Quick Start

<Tabs>
  <Tab title="cURL">
    ```bash
    curl -X POST "http://localhost:8000/predict/easyocr/english-light" \
      -F "file=@document.jpg"
    ```
  </Tab>

  <Tab title="Python">
    ```python
    import requests

    with open('document.jpg', 'rb') as f:
        response = requests.post(
            'http://localhost:8000/predict/easyocr/english-light',
            files={'file': f}
        )

    results = response.json()
    for item in results:
        print(f"{item['text']} (confidence: {item['confidence']:.2f})")
    ```
  </Tab>

## Python SDK

Use EasyOCR models directly in Python without the HTTP server.

### Quick Start

```python
from mozo import ModelManager
import cv2

# Initialize manager
manager = ModelManager()

# Load model
model = manager.get_model('easyocr', 'english-light')

# Load image and extract text
image = cv2.imread('document.jpg')
results = model.predict(image)

# Access text results
for item in results.detections:
    print(f"Text: {item.text}")
    print(f"Confidence: {item.confidence:.2%}")
    print(f"Location: {item.bbox}")
```

### Using Different Variants

```python
# Fast English OCR
light_model = manager.get_model('easyocr', 'english-light')
results = light_model.predict(image)

# High-accuracy English OCR
full_model = manager.get_model('easyocr', 'english-full')
results = full_model.predict(image)

# Multi-language support
multilingual = manager.get_model('easyocr', 'multilingual')
results = multilingual.predict(image)

# Chinese documents
chinese_model = manager.get_model('easyocr', 'chinese')
results = chinese_model.predict(image)
```

### Filtering and Processing Results

```python
# Filter by confidence
high_conf_text = [
    item for item in results.detections
    if item.confidence > 0.8
]

# Extract just the text
text_only = [item.text for item in results.detections]
full_text = '\n'.join(text_only)
print(full_text)

# Group text by vertical position (approximate lines)
sorted_by_y = sorted(results.detections, key=lambda d: d.bbox[1])
for item in sorted_by_y:
    print(item.text)
```

### Batch Document Processing

```python
import os

model = manager.get_model('easyocr', 'english-light')
documents = []

for filename in os.listdir('documents/'):
    if filename.endswith(('.jpg', '.png', '.pdf')):
        image = cv2.imread(os.path.join('documents/', filename))
        results = model.predict(image)

        text = '\n'.join([item.text for item in results.detections])
        documents.append({
            'filename': filename,
            'text': text,
            'word_count': len(text.split())
        })

print(f"Processed {len(documents)} documents")
```

### Visualization

```python
import pixelflow as pf

# Annotate image with detected text regions
annotated = pf.annotate.draw_detections(
    image=image,
    detections=results,
    labels=True,  # Show recognized text
    boxes=True    # Show bounding boxes
)

cv2.imwrite('annotated.jpg', annotated)
```

### Search and Extract

```python
# Find specific text patterns
results = model.predict(image)

# Search for invoice number
for item in results.detections:
    if 'INV-' in item.text.upper():
        print(f"Found invoice number: {item.text}")

# Extract amounts (simple regex example)
import re
for item in results.detections:
    # Look for currency patterns
    if re.search(r'\$[\d,]+\.?\d*', item.text):
        print(f"Found amount: {item.text}")
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Low accuracy on handwriting">
    EasyOCR is optimized for printed text. For handwriting, consider specialized models.
  </Accordion>

  <Accordion title="Slow on large images">
    Resize images before processing:
    ```python
    from PIL import Image
    img = Image.open('large.jpg')
    img.thumbnail((1920, 1080))
    img.save('resized.jpg')
    ```
  </Accordion>

  <Accordion title="Missing text">
    - Ensure sufficient image quality (min 150 DPI recommended)
    - Check text contrast against background
    - Try english-full for better accuracy
  </Accordion>
</AccordionGroup>
